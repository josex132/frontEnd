SportStock Sistema de Gestión de Inventario Deportivo diseñado para optimizar el control de productos, stock y usuarios en una interfaz moderna y rápida.

Funciones Clave Dashboard: Resumen de valor total y productos.

Alertas: Notificación visual de stock bajo.

Inventario: Registro y control de artículos deportivos.

Usuarios: Gestión de roles (Admin/Empleado).

Navegación SPA: Cambio entre secciones sin recargar la página.

Tecnologías Frontend: HTML5, CSS3 (Grid y Flexbox).

Diseño: UI/UX limpio con tipografía Quicksand.
